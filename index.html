WELCOME TO BEATBOP

BEATBOP is a browser-based step sequencer and groovebox. Open index.html in any modern browser — no installation or build step. All logic lives in the js/ folder (14 modules).

Quick Start

  • Press ▶ Play to hear the demo pattern
  • Click grid cells to toggle steps on/off
  • Double-click a step to edit pitch, velocity, and duration
  • Right-click for more options (long-press on touch devices)

BPM & Swing

Open ⋯ More options in the header to set BPM (20–300) and Swing. BPM is applied immediately to the transport. Swing delays odd-numbered steps for a shuffle feel (0% = straight, 50–70% = hip-hop/jazz groove).

Right-click any track label → Load Sample. Supports .wav, .mp3, .ogg, .aiff.

STEP GRID

Toggling Steps

In STEPS mode with ADD (+) selected, click any cell to activate it. Click an active cell to turn it off. Drag across cells to paint/erase in bulk.

Reading a Step Cell

  • Center text — note name (e.g. C4, D#3, G2)
  • Top-left glyph — duration: ♬ = 16th, ♪ = 8th, ♩ = quarter, 𝅗𝅥 = half
  • Bottom bar — velocity fill (wider = louder)
  • Top-right dot — per-step swing is active

Step Editor (Detailed Control)

Right-click or double-click any step to open the Step Editor popup:

PARAMETERRANGEEFFECT
Velocity0–127Note loudness and brightness
Duration1–16 (16ths)How long the note sustains after the step triggers
Pitch offsetsemitonesRaises/lowers note relative to track's base pitch; note name shown live
Swing0–100%Individual step delay, independent of global swing

Beat Markers

Step numbers highlighted in bold (1, 5, 9, 13) mark the four quarter-note beats of a 4/4 bar.

Ties

A tie links a step to the previous note, holding it through both steps for a longer sustained note. Right-click a step → Toggle Tie.

PAGES & ARRANGEMENT

Pages let you chain multiple 16-step patterns together. During playback, BEATBOP plays Page 1, then Page 2, then Page 3… looping back to Page 1. This creates evolving compositions up to 256 steps long (16 pages × 16 steps).

Navigation

  • ‹ / › arrows in the Timeline — go to previous/next page
  • Click the minimap — jump to any page directly
  • ← / → arrow keys — navigate pages from keyboard
  • › on the last page — automatically creates a new page

Page indicator

The Toolbar shows P 1/1 (current page / total). Click it to open the Page menu (New, Copy, Paste, Clear, Delete). The green frame in the timeline minimap marks the page you're viewing; a red frame marks the page currently playing.

Page menu actions

Click the P 1/1 button in the Toolbar to access:

BUTTONACTION
NEW PAGEAdd a page after the current one (pre-filled with current page's content)
COPY PGCopy all track data from this page to the clipboard
PASTE PGPaste clipboard page data onto the current page
CLEAR ALLErase every step on the current page across all tracks
DEL PAGERemove the current page (minimum 1 page always remains)

The Timeline Minimap

  • Each horizontal strip = one track; colored blocks = active steps
  • Block brightness indicates step velocity
  • Vertical dividers separate pages
  • Click any region to jump immediately to that page

NOTE EDITOR (PIANO ROLL)

A full piano roll interface for precise melodic and harmonic editing. In Notes mode the grid shows each step as a column of 8 rectangles (scale degrees 1–8). Click any track row to open the piano roll for that track.

Layout

  • Piano keyboard (left strip) — C1 to C8; click any key to preview that pitch
  • Time ruler (sticky top) — step numbers with green beat markers
  • Note grid — semitone rows, step columns; black-key rows are slightly darker
  • Note blocks — color = track color; opacity = velocity; width = duration
  • Velocity strip (bottom) — bar chart; drag bars vertically to adjust velocity

Tools

TOOLKEYACTION
✎ DrawDClick empty cell to add note; click again to remove. Drag to paint across multiple steps.
✕ EraseETap or drag over notes to remove them.

CLEAR removes all notes from the current track.

Quantization Snap

Click SNAP 1/16 in the toolbar to cycle: 1/16 → 1/8 → 1/4 → 1/2. Notes snap to the nearest grid value when placed or moved.

Editing Velocity

In the Velocity strip at the bottom of the editor, click and drag any bar upward to increase velocity, downward to decrease it.

All Note Editor changes sync immediately to the step sequencer. Closing the editor saves your work automatically.

EDITOR MODES

The Steps / Notes segmented control in the Toolbar switches between two editing modes. Each mode reveals a different set of action buttons.

STEPS Mode

The default mode — edit individual steps on the grid.

BUTTONACTION
+ (Add)Click or drag to add steps
− (Delete)Click or drag to remove steps
⧉ (Copy)Copy a step's data to clipboard
⧫ (Paste)Paste clipboard data onto a step

NOTES Mode

Shows a pitch visualization for each track. Click any track row to open the piano roll Note Editor for that track. Buttons switch to Copy Track and Paste Track.

Page menu

The P 1/1 button (right side of Toolbar) opens the page menu: New, Copy, Paste, Clear, Delete.

Undo / Redo

The ↩ / ↪ buttons (far right of Toolbar) and Ctrl+Z / Ctrl+Y undo and redo edits. Click the ▸ chevron in the Console bar to expand the full action history — click any entry to revert to that state (you'll be asked to confirm).

Context menus

Step cell right-click — volume slider, Edit Step…, Toggle Tie, Copy/Paste Track, Import MIDI, Export MIDI, Load Sample, Delete Track.

Track label right-click or ⚙ — Copy/Paste track, Import MIDI, Export MIDI, Mute, Solo, Load Sample, Staff Notation, Delete Track.

EXPORT & SAVE

Save Project

Open ⚙ Settings and click SAVE PROJECT. Downloads hexbeat_project.json containing your full composition — BPM, swing, all tracks, all steps, all pages.

Load Project

Open ⚙ Settings → LOAD PROJECT and select a previously saved .json file. The current session is replaced.

Note: Audio samples are not saved in the project file — only their filenames. You'll need to reload WAV files after opening a saved project.

Export Track MIDI

Right-click any track label or step → Export Track MIDI. Downloads a standard .mid Type-0 file for the current page. Import directly into Ableton Live, Logic, GarageBand, FL Studio, or any DAW.

Record Audio

  • Press ● Record (or R) to start capturing
  • Press ▶ Play to hear and record your pattern
  • Press ● Record again to stop
  • Click ↓ SAVE AUDIO to download as a .webm audio file

Export MIDI

Settings → EXPORT MIDI downloads all tracks as one .mid file. To export a single track, right-click the track label or a step → Export MIDI.

Import MIDI

Settings → MIDI IMPORT replaces all tracks with the first track from a MIDI file. To import into one track only, right-click the track label or a step → Import MIDI.

Staff Notation

Right-click any track (or click ⚙) → Export Staff Notation to view a canvas-rendered musical staff for that track.
