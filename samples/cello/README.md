# Cello sample pack

Place `.mp3` or `.wav` files in this folder to use the **Cello** variant (Instrument → Strings → Cello).

Default probe notes:

- `C2.mp3` / `C2.wav`
- `C3.mp3` / `C3.wav`

Optional override in the browser console before loading:

`window.BeatBopCelloSampleNotes = ['C2', 'G2', 'C3']`
