# 707 kit samples

Place `.wav` or `.mp3` files in this folder to use the **707** drum kit (Instrument → Drums → Kit: 707). Each voice maps to one file; the app looks for `.wav` first, then `.mp3` for each name.

| Voice       | Filename              |
|------------|------------------------|
| Kick       | kick.wav or kick.mp3   |
| Snare      | snare.wav or snare.mp3 |
| Low Tom    | low_tom.wav / .mp3     |
| Mid Tom    | mid_tom.wav / .mp3     |
| Hi Tom     | hi_tom.wav / .mp3      |
| Rimshot    | rimshot.wav / .mp3     |
| Cowbell    | cowbell.wav / .mp3     |
| Clap       | clap.wav / .mp3        |
| Closed HH  | closed_hh.wav / .mp3   |
| Open HH    | open_hh.wav / .mp3     |
| Crash      | crash.wav / .mp3       |
| Ride       | ride.wav / .mp3        |
| Tambourine | tambourine.wav / .mp3  |

Serve the app from the project root (e.g. `npm start`) so paths resolve correctly.
