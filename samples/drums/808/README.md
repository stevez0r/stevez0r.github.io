# 808 kit samples

Place `.wav` files in this folder to use the **808** drum kit (Instrument → Drums → Kit: 808).

Only **`.wav`** files are shown as Voice options in the UI for this kit. If a file is missing, its voice button will not appear.

| Voice       | Filename        |
|------------|-----------------|
| Kick       | kick.wav        |
| Snare      | snare.wav       |
| Low Tom    | low_tom.wav     |
| Mid Tom    | mid_tom.wav     |
| Hi Tom     | hi_tom.wav      |
| Rimshot    | rimshot.wav     |
| Cowbell    | cowbell.wav     |
| Clap       | clap.wav        |
| Closed HH  | closed_hh.wav   |
| Open HH    | open_hh.wav     |
| Crash      | crash.wav       |
| Ride       | ride.wav        |
| Tambourine | tambourine.wav  |

Serve the app from the project root (e.g. `npm start`) so paths like `samples/drums/808/kick.wav` resolve correctly.
