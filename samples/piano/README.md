# Piano sample pack

Place `.mp3` or `.wav` files in this folder to use the **Piano** variant (Instrument → Keys → Piano).

Default probe notes:

- `C3.mp3` / `C3.wav`
- `C4.mp3` / `C4.wav`

Optional override in the browser console before loading:

`window.BeatBopPianoSampleNotes = ['C2', 'C3', 'E3', 'G3', 'C4']`
