# Guitar acoustic sample pack

Place `.mp3` or `.wav` files in this folder to use the **Acoustic Sample** variant (Instrument → Guitar → Acoustic Sample).

Default probe notes:

- `C3.mp3` / `C3.wav`
- `C4.mp3` / `C4.wav`

Optional override in the browser console before loading:

`window.BeatBopGuitarAcousticSampleNotes = ['C3', 'E3', 'G3', 'C4']`
