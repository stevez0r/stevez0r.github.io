# Noto Music Font

This directory contains the locally hosted Noto Music font files.

## Font Files Required

Place the following Noto Music font files in this directory:

- `NotoMusic-Regular.woff2` - WOFF2 format (recommended, best compression)
- `NotoMusic-Regular.woff` - WOFF format (fallback for older browsers)
- `NotoMusic-Regular.ttf` - TrueType format (fallback)

## Getting Noto Music Font

Download Noto Music from:
https://fonts.google.com/noto/specimen/Noto+Music

Or from the official Noto repository:
https://github.com/notofonts/noto-fonts

Extract the font files and place them in this directory.
